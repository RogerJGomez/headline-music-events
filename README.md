## Setup

To run this project, install it locally using npm:

  

```

$ cd ../project_folder

$ npm install

$ npm start

```

## General info

  

Find incoming events of your favorite music artists, you can search them by name. Events are tracked using the Bandsintown API

 

## Technologies

Project is created with:

* ReactJS
* React-MDL
* Bandsintown API
